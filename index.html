foxloves

A dependency-free, themeable UI widget library for LÖVE (love2d). 24 widgets sharing one input and drawing lifecycle.

Controls, containers, and overlays that share one theme and one input/draw lifecycle — no dependencies, pure Lua on LÖVE 11.x. This site is the manual; the design system itself lives in its own repository.

View the foxloves source on GitHub →

Install

Copy the foxloves/ folder onto your require path, then:

local fox = require("foxloves")

Quick start

local fox = require("foxloves")
local ui = fox.Root.new()
ui:add(fox.Button.new{ x = 40, y = 40, w = 120, h = 34, label = "Greet" })

function love.update(dt) ui:update(dt) end
function love.draw()    ui:draw() end
