ToastHost

Container / Overlay

ToastHost widget: transient, stacked notification messages. Add it to a fox.Root (root:add). Push messages with: host:show("Saved.", { kind = "success", duration = 2 }) kind is info | success | warning | error and colors the left stripe; it reads theme.color[kind] and falls back to accent. show() returns a handle you can pass to host:dismiss(handle) to fade a toast out early. The host never captures input, so add it after the widgets it floats over.

Appearance

Capabilities

Display only

Options

OptionTypeDefaultDescription
corner string "br" tl | tr | bl | br (which corner toasts stack in)
gap number 8 pixels between stacked toasts
margin number 12 pixels from the screen edges
width number 260 toast box width
duration number 3 default seconds a toast stays before fading out
max number 4 most toasts kept at once (oldest dropped past this)
theme table <default theme> Theme table override; falls back to the default.

Example

local fox = require("foxloves")
local toast = fox.ToastHost.new{
  x = 40, y = 40,
}

Source

function ToastHost.new(opts)
  opts = opts or {}
  local self = setmetatable({}, ToastHost)
  self.corner = opts.corner or "br"
  self.gap = opts.gap or 8
  self.margin = opts.margin or 12
  self.width = opts.width or 260
  self.duration = opts.duration or 3
  self.max = opts.max or 4
  self.theme = opts.theme or defaultTheme
  self.toasts = {}  -- newest last
  return self
end
