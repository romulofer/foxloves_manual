Avatar

Control

Avatar / Image widget: a framed image, circle or rounded square, with an initials fallback when no image is given. Non-interactive. Exposes self.w/self.h (= size) and :measure() so layout containers can place it.

Appearance

Capabilities

Display only

Options

OptionTypeDefaultDescription
x number 0 X position, in pixels.
y number 0 Y position, in pixels.
size number 40 square; w = h = size
image any — love Image; cover-scaled and cropped to the frame
name any nil derives fallback initials ("Red Fox" -> "RF")
initials any initialsFromName(opts.name) explicit override; wins over name
shape string "circle" "circle" | "rounded"
color any — fallback fill when there is no image; default accent
textColor any — initials color; default theme.color.bg
theme table <default theme> Theme table override; falls back to the default.

Example

local fox = require("foxloves")
local avatar = fox.Avatar.new{
  x = 40, y = 40,
}

Source

function Avatar.new(opts)
  opts = opts or {}
  local self = setmetatable({}, Avatar)
  self.x = opts.x or 0
  self.y = opts.y or 0
  self.size = opts.size or 40
  self.w, self.h = self.size, self.size
  self.image = opts.image
  self.initials = opts.initials or initialsFromName(opts.name)
  self.shape = opts.shape or "circle"
  self.color = opts.color
  self.textColor = opts.textColor
  self.theme = opts.theme or defaultTheme
  return self
end
