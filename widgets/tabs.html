Tabs

Container / Overlay

Tabs widget: switch between panels via a header row. Draws a row of clickable tab labels; below it, the selected tab's `panel` (any widget, typically a fox.Panel) is drawn and receives input. Position each panel below the header yourself. Switching fires onChange(index).

Appearance

Capabilities

Focusable Hover leftrighthomeend

Options

OptionTypeDefaultDescription
x number 0 X position, in pixels.
y number 0 Y position, in pixels.
w number 300 Width, in pixels.
headerH number 32 Header row height, in pixels.
tabs table {} Tab specs: { label, panel }.
selected number 1 Index of the selected item.
onChangefn function — Called when the value changes.
theme table <default theme> Theme table override; falls back to the default.

Example

local fox = require("foxloves")
local tabs = fox.Tabs.new{
  x = 40, y = 40,
  w = 200,
}

Source

function Tabs.new(opts)
  opts = opts or {}
  local self = setmetatable({}, Tabs)
  self.x = opts.x or 0
  self.y = opts.y or 0
  self.w = opts.w or 300
  self.headerH = opts.headerH or 32
  self.tabs = opts.tabs or {}
  self.selected = opts.selected or 1
  self.onChange = opts.onChange
  self.theme = opts.theme or defaultTheme
  self.hoverTab = nil  -- header segment under the cursor, or nil
  self.focusable = true
  return self
end
