Spinner

Control

Spinner widget: an animated busy indicator for unknown-duration work. Non-interactive. A ring of dots fades in a rotating trail; advance it by calling update(dt). Exposes self.w/self.h and :measure() (both = size) so layout containers can place it.

Appearance

Capabilities

Display only

Options

OptionTypeDefaultDescription
x number 0 X position, in pixels.
y number 0 Y position, in pixels.
size number 24 bounding box (square) in pixels
dots number 8 number of dots around the ring
speed number 1 revolutions per second
color any — dot color override (table); default theme.color.accent
theme table <default theme> Theme table override; falls back to the default.

Example

local fox = require("foxloves")
local spinner = fox.Spinner.new{
  x = 40, y = 40,
}

Source

function Spinner.new(opts)
  opts = opts or {}
  local self = setmetatable({}, Spinner)
  self.x = opts.x or 0
  self.y = opts.y or 0
  self.size = opts.size or 24
  self.dots = opts.dots or 8
  self.speed = opts.speed or 1
  self.color = opts.color
  self.theme = opts.theme or defaultTheme
  self.w, self.h = self.size, self.size
  self.phase = 0   -- rotation, in [0, 1)
  return self
end
