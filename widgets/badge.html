Badge

Control

Badge / Tag / Chip widget: a small rounded label for counts or status. Self-sizes to its text (pill-shaped); optionally removable as a chip. Non-interactive unless `removable`. Exposes computed self.w/self.h and a :measure() returning w,h so layout containers can place it. `text` is mutable via setText, which re-measures.

Appearance

Capabilities

Hover

Options

OptionTypeDefaultDescription
x number 0 X position, in pixels.
y number 0 Y position, in pixels.
text string "" Displayed text.
color any — fill override (table); default theme.color.accent
textColor any — label color override; default theme.color.bg
removable boolean false draw a × hitbox on the right; fires onRemove on click
onRemovefn function — Called when removed.
theme table <default theme> Theme table override; falls back to the default.

Example

local fox = require("foxloves")
local badge = fox.Badge.new{
  x = 40, y = 40,
  text = "",
}

Source

function Badge.new(opts)
  opts = opts or {}
  local self = setmetatable({}, Badge)
  self.x = opts.x or 0
  self.y = opts.y or 0
  self.text = opts.text or ""
  self.color = opts.color
  self.textColor = opts.textColor
  self.removable = opts.removable or false
  self.onRemove = opts.onRemove
  self.theme = opts.theme or defaultTheme
  self.removeHovered = false
  self.removePressed = false
  measureSelf(self)
  return self
end
